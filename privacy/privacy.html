Privacy Policy

Last updated: 9 July 2026

This page explains what technical data is processed when you visit my portfolio site, why it is processed, and how your privacy is protected. I do not use advertising tracking cookies, nor do I sell or distribute personal user information.

1. Data Controller

Anubhab Pathak, independent developer and designer. As this is a personal portfolio website, I act as the sole controller of any minimal data handling mentioned below.

2. Infrastructure & Hosting

This website is hosted on GitHub Pages (provided by GitHub Inc., USA). GitHub receives standard server request metadata (such as your IP address, browser type, and access timestamp) necessary to handle platform security, delivery of the website assets, and infrastructure diagnostic logs. This processing aligns with the GitHub Privacy Statement.

3. Analytics & Visitor Insights

To evaluate website traffic ethically without compromising user identity, this site utilizes Umami Analytics (Umami Cloud free tier). This tool is built entirely for privacy compliance:

  • It does **not** use tracking cookies.
  • It does **not** collect or store personal identities.
  • It does **not** track visitors across different websites or days.

The analytics tool stores aggregated performance data (e.g., country location, device type, specific page views) to help me monitor site performance without profiling human visitors.

4. Third-Party Assets & Content Delivery Networks (CDNs)

To optimize performance, typography, and visual interface rendering, this website requests design assets dynamically from external networks:

  • Google Fonts: Typographic files are requested directly from Google LLC asset distribution servers.
  • Font Awesome: Graphical icon kits are retrieved from Font Awesome’s networks.

When your web browser loads this site, it makes an automated asset fetch request to these companies. During this network event, your IP address is visible to them to facilitate the download file transfer.

5. Media and Media Ownership

The background video implemented on this portfolio was sourced from Pinterest (Credits to: Видеофон). To prioritize visitor privacy, this video asset has been downloaded locally and is served directly from the website hosting codebase framework. No third-party network connections, tracking elements, or behavioral tracking scripts from Pinterest are executed when you access this site.

6. International Data Transfers

The primary hosting infrastructure (GitHub) and external content delivery services operate servers within the United States. Where request metadata (like your IP address during font downloading) passes outside your local region, these providers rely on legal standard safeguards, such as the EU-U.S. Data Privacy Framework or standard contractual clauses, to maintain data safety.

7. Changes to This Notice

This privacy notice may be modified over time to maintain consistency with package tools or network changes. Updates take effect immediately when published with a fresh "Last Updated" timestamp at the top of this page.
